# 🛡️ A股 ETF 国家队三因子监控看板

> 追踪中央汇金 ETF 操作信号，三因子模型：量能×50% + 方向×20% + 份额×30%

## 数据源（零第三方依赖）
- K线行情：腾讯财经 HTTP API（主）+ 新浪财经（备）
- ETF份额：东方财富实时 API f84 字段（双入口 fallback）
- 本地存储：SQLite（etf_history.db）
