# Ralph Round 4 Factor Check — 量能因子(P50%)验证

## 代码公式

`etf_v7_threefactor.py` 中量能因子的实际计算：

1. 在 `analyze_all()` 内，对每个交易日：
   - `v = d["v"] / 10000`
   - `pv = [data[j]["v"] / 10000 for j in range(i - 20, i)]`
   - `ma = sum(pv) / 20`
   - `vr = v / ma`
   - `vp = vprob(vr)`
2. `vprob(r)` 是分段线性映射，不是统计分位数：
   - `r < 0.5`: `r / 0.5 * 5`
   - `0.5 <= r < 1.0`: `5 + (r - 0.5) / 0.5 * 12`
   - `1.0 <= r < 1.3`: `17 + (r - 1) / 0.3 * 18`
   - 后续更高倍率继续映射到 100。
3. 综合分权重：
   - 份额因子可用时：`cp = vp * 0.5 + dp * 0.2 + sp * 0.3`
   - 份额因子不可用时：`cp = vp * 0.7 + dp * 0.3`
